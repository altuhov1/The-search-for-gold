## Технологии
- **Язык**: C (ANSI C / C99)
- **Библиотека**: [`ncurses`](https://invisible-island.net/ncurses/) — управление терминальным вводом/выводом, псевдографика
- **Сборка**: `make` (без внешних зависимостей, кроме `libncurses-dev`)
- **Платформа**: Linux / macOS (работает в любом UTF-8 терминале с поддержкой ANSI)

```bash
# Клонировать репозиторий

cd game_2d

# Собрать (требуется gcc и ncurses)
